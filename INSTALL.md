# INSTALLATION GUIDE #

## Compile yourselfe ##
1. Download the source files
2. In the folder of the project you should find two files that named Build (`Build.sh` and `Build.bat`)
    - On Windows, start `Build.bat`
    - On Linux, start `Build.sh`
3. You can run the calculator that you can find in the `Binary` folder

## Install on Linux (.deb package) ##
1. Download the `IP-Calculator_v1.0.0.deb` file
2. Install it
    ```bash
    $ dpkg -i IP-Calculator_v1.0.0.deb 
    ```
3. You can run the calculator in the terminal

## Install on Windows ##
1. Donwload the `ip-calculator.exe` file
2. You can run the calculator in the Command Prompt
